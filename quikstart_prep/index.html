Before Running

1) Preparation

After Downloading and unzipping BridgePRS in a suitable directory (referred to as $BridgeDir) on your machine you will observe the following file structure: 

 bridgePRS data/ src/ LICENSE README.md

Using the terminal, type the following command from within the directory:

    chmod +x bridgePRS
to make the bridgePRS file executable

2) Requirements

Next, confirm that the required libraries and binaries are installed and available by following the detailed instructions in Requirements. Alternatively, you can type: 

./bridgePRS check requirements

to bridgePRS to check your system and prompt you to install missing requires.

3) Input Data

BridgePRS requires input population data to be passed using command line arguments or population configuration files which tell the program where population specific data (sumstats, phenotypes, etc) is located. Locate the toy data directory $Bridgedir/data where population config files for the tutorial are located: 

 .
 plink_files/  
 afr_bin_files/ afr_cont_data/
 eur_bin_files/ eur_cont_data/
 afr.bin.config     afr.cont.config  
 eur.bin.config     eur.cont.config  
 options.specifications

In this tutorial the primary population is of "AFR" ancestry and the model population is of "EUR" ancestry:

  • Primary Pop Config Files: (afr.bin.config, afr.cont.config)
  • Model Pop Config Files: (eur.bin.config, eur.cont.config)
  • Options Specification File: (options.specifications)
